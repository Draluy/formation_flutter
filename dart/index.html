pour les developpeurs Java

Introduction

Dart est le langage de programmation pour Flutter, le fw de Google pour construire des applications natives pour mobile, web et desktop, a partir d'une seule codebase

Cette presentation va vous presenter Dart, en mettant l'accent sur les features que les developpeurs Java peuvent ne pas connaitre.

Au fur et a mesure de la presentation vous pouvez jouer avec Dartpad, pour tester les differentes features du langage.

Creer une classe


class Bicycle {
  int cadence;
  int speed;
  int gear;
}

void main() {...}
  • main() ou main(List<String> args).
  • main est au niveau le plus haut, et peut etre en dehors d'une classe. En Dart, des variables et fonctions peuvent etre declarees en dehors des classes.
  • Tout est public par defaut.

Ajoutons un constructeur


Bicycle(this.cadence, this.speed, this.gear);
  • Le constructeur n'a pas de body, ce qui est valide.
  • Le ; a la fin est important
  • `this` en tant que parametre est un raccourci pour assigner des valeurs aux attributs.

Ajoutons un constructeur


Bicycle(this.cadence, this.speed, this.gear);

Est equivalent a: 


Bicycle(int cadence, int speed, int gear) {
  this.cadence = cadence;
  this.speed = speed;
  this.gear = gear;
}

Instancier une classe


void main() {
  var bike = Bicycle(2, 0, 1);
  print(bike);
}
  • Le mot cle `new` est optionnel
  • Si une variable ne doit pas changer, utiliser `final` au lieu de `new`

Afficher une classe

  • Toutes les classes ont une methode `toString()`.
  • Pour la redefinir: 
    
@override
String toString() => 'Bicycle: $speed mph';
  • On peut remarquer les string literals et la notation fat arrow pour les fonctions.

Ajouter une variable en lecture seule 1/2


int _speed = 0;

int get speed => _speed;
  • Les variables non initialisees sont null.
  • Le compilateur Dart verifie que les variables commencant avec un _ sont privees.

Ajouter une variable en lecture seule 2/2


int _speed = 0;

int get speed => _speed;
  • Par defaut Dart cree des getters/setters pour les attributs publics. Il n'est pas necessaire de les creer sauf dans le cas present.
  • On accede aux champs via `bike.cadence`. Cette api reste la meme, getter ou pas.

Utiliser des parametres optionnels 1/2


Rectangle({this.origin = const Point(0, 0), this.width = 0, this.height = 0});
  • Les parametres nommmes sont entre {}.
  • this.origin, this.width, et this.height sont des parametres nommes optionnels.

Utiliser des parametres optionnels 2/2

On peut alors utiliser le constructeur comme suit:


main() {
  print(Rectangle(origin: const Point(10, 20), width: 100, height: 200));
  print(Rectangle(origin: const Point(10, 10)));
  print(Rectangle(width: 200));
  print(Rectangle());
}

Creer une factory 1/2

Possibilite 1: fonction de haut niveau


Shape shapeFactory(String type) {
  if (type == 'circle') return Circle(2);
  if (type == 'square') return Square(2);
  throw 'Can\'t create $type.';
}

Note: Les exceptions heritent de `Exception`

Creer une factory 2/2

Possibilite 2: utiliser le mot-cle `factory`


abstract class Shape {
  factory Shape(String type) {
    if (type == 'circle') return Circle(2);
    if (type == 'square') return Square(2);
    throw 'Can\'t create $type.';
  }
  num get area;
}


 final circle = Shape('circle');

Implementer une interface

Il n'y a pas d'interfaces en Dart car chaque classe definit une interface.


class Circle implements Shape {
  final num radius;
  Circle(this.radius);
  num get area => pi * pow(radius, 2);
}


class CircleMock implements Circle {}

Programmation fonctionnelle

En Dart on peut:

  • Passer des fonctions comme arguments.
  • Assigner une fonction a une variable.
  • Utiliser le currying.
  • Utiliser des fonctions anonymes (lambdas)

Programmation fonctionnelle

Exemple imperatif


String scream(int length) => "A${'a' * length}h!";
final values = [1, 2, 3, 5, 10, 50];
for (var length in values) {
  print(scream(length));
}
