Introduction

Dans cette partie nous allons voir comment coder un hello world en Flutter

lib/main.dart


import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Welcome to Flutter'),
        ),
        body: Center(
          child: Text('Hello World'),
        ),
      ),
    );
  }
}

Observations

  • L'application etend StatelessWidget qui fait de l'application un widget. En Flutter, tout est un widget, meme l'alignment, le padding, and le layout general.
  • Le widget Scaffold, de la librairie Material cree une app bar, un titre et une propriete body qui contient l'arborescnce des widgets pour la page d'accueil.
  • Le but d'un widget est de proposer une methode build qui decrit comment afficher le widget en composant avec d'autres widgets de plus bas niveau.

Widgets stateful

Les widgets stateless sont immutables, cad que leurs proprietes ne peuvent pas changer.

Les widgets stateful ont un etat qui peut changer pendant la duree de vie du widget.

Implementer un widget stateful necessite deux choses:

  1. Une classe StatefulWidget qui cree une instance
  2. Une classe State

Creer un State


class RandomWords extends StatefulWidget {
  @override
  RandomWordsState createState() => RandomWordsState();
}
        

class RandomWordsState extends State<RandomWords> {
   @override
  Widget build(BuildContext context) {
    final wordPair = WordPair.random();
    return Text(wordPair.asPascalCase);
  }
}
